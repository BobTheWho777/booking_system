# Система бронирования номеров «Олимп»

Система бронирования и учёта номеров для мини-гостиницы или хостела.

## Особенности

### 🏛️ Дизайн в стиле Древней Греции
- Мраморные текстуры и градиенты
- Золотые акценты и декоративные элементы
- Классические шрифты (Cinzel, Cormorant Garamond, Philosopher)
- Средиземноморская цветовая палитра

### ✨ Функциональность

#### Публичная часть:
- Главная страница с поиском номеров по датам
- Оформление бронирования онлайн
- Автоматический расчёт стоимости

#### Админ-панель:
- Управление комнатами (CRUD)
- Управление типами номеров (CRUD)
- База гостей (CRUD)
- Просмотр бронирований
- Отчёты по загрузке и доходам

### 🛠️ Технологии
- PHP 8+
- MySQL
- Vanilla JavaScript
- CSS3 с кастомной грекской темой

## Установка

1. Скоируйте файлы в директорию веб-сервера
2. Создайте базу данных `hotel_db`
3. Импортируйте схему БД (файл `database/schema.sql`)
4. Настройте подключение к БД в `config/db.php`
5. Создайте пользователя-администратора:
   ```sql
   INSERT INTO users (username, password_hash, role) VALUES
   ('admin', '$2y$10$92IXUNpkjO0rOQ5byMi.Ye4oKoEa3Ro9llC/.og/at2.uheWG/igi', 'admin');
   ```
   (пароль: `password`)
